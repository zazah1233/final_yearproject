Manual Retraining Process

To retrain the models with custom data, follow these steps:

  1. Prepare a CSV file with at least 500 patient records
  2. Ensure the CSV contains the required 22 clinical features plus a 'typhoid' column (0 for negative, 1 for positive)
  3. Place the CSV file in the data/ directory
  4. Run: python ml/train.py from the project root
  5. The script will use your CSV and retrain the models
  6. Models will be saved to data/rf_model.joblib and data/dt_model.joblib
